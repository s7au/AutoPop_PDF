Hi this was my first time using PHP... and really HTML, and originally was supposed to be integrated into a SugarCRM module. Unfortunately SugarCRM blacklists important functions in its module loader so installing such a module on a Sugarondemand server became impossible for someone like me.

The point is to autopopulate an existing PDF file with personal information. Say I have a form that I have to fill 500 times but with different information each time. As long as I have a file that denotes word placement I can use this to fill out the form efficiently.

Please read the Directions tab. As an afterthought the jQuery on this page was done by a friend.

Code has not been synced with UI so nothing works atm...

This needs two libraries: FPDI and TCPDF. Please extract these two libraries into the root of your primary domain. I did not include them because I think there might be a law against that. Or there should be.

Create a text file according to the template on "Input Template" tab. Keep the first line as the code will read starting from the second line. Also remember to separate your columns with white space. This really was not made to be bug proof. If entirely not obvious, here are the column descriptions:

The first column is the page number.

The second and third column denote the x and y coordinate of the text. According to google a typical paper has dimensions 210 mm X 297 mm. Say (x,y) denote your coordinate on the page. Top left is (0,0), top right is (210,0), bottom left is (0,297), and bottom right is (210,297). Hopefully that makes sense.

The final column is the field tag. Currently I have the following tags: First_Name, Last_Name, Street, City, Province, Country, Postal_Code, Day, Month, Year. If I have time will see if I can implement custom tags. Please preserves cases and underscores!

Finally go to Autopopulate tab, input the information you want in each field (e.g. Bob in First_Name), upload your PDF and your Input Template, click the Populate button and... whala! If I finish this something will happen.
